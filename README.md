# 快速开始指南 / Quick Start Guide

## 如何运行游戏 / How to Run

1. 在浏览器中打开 `index.html`
   - Open `index.html` in any modern web browser

2. 游戏自动加载，棋盘显示初始位置
   - Game loads automatically with initial board position

## 游戏规则 / Game Rules

### 棋盘布局 / Board Layout
- **棋盘:** 9×10 (宽×高)
- **河界:** 第 4-5 行（楚河汉界）
- **宫殿:**
  - 红方: 第 7-9 行，列 3-5
  - 黑方: 第 0-2 行，列 3-5

### 棋子 / Pieces

| 中国象棋 (红方) | 符号 | 国际象棋 (黑方) | 符号 |
|---|---|---|---|
| 将 | 將 | 王 | ♔ |
| 士 | 士 | 后 | ♕ |
| 象 | 象 | 象 | ♗ |
| 马 | 馬 | 马 | ♘ |
| 车 | 車 | 车 | ♖ |
| 砲 | 砲 | - | - |
| 兵 | 兵 | 兵 | ♙ |

### 特殊规则 / Special Rules

1. **将帅照面** - 两方将/王不能在同一列且中间无子
2. **象不过河** - 红方象不能越过河界
3. **炮需炮架** - 炮吃子需隔一个棋子作为"炮架"
4. **兵升变** - 黑方兵到达第 9 行时升变为后
5. **将军** - 若己方将/王被攻击则为将军
6. **将死** - 将军且无任何合法走法则为将死

## 操作方法 / How to Play

### 移动棋子 / Move Pieces
1. **点击** 一个红方棋子（你的棋子）
   - 绿色方块显示可走位置
2. **点击** 绿色方块移动棋子
3. 自动切换到黑方
4. 重复步骤 1-3 操作黑方（用国际象棋规则）

### UI 指示 / UI Indicators
- 🟢 **绿色**: 可走位置
- 🟠 **橙色**: 上一步移动的起点和终点
- 🔴 **红色**: 将军状态

### 重新开始 / Reset Game
- 点击 "重新开始" 按钮重新开始游戏

## 游戏状态 / Game Status

状态栏显示当前回合和游戏状态：
- `当前回合: 红方 (中国象棋)` - 红方轮到
- `当前回合: 黑方 (国际象棋)` - 黑方轮到
- `当前回合: ... - ⚠️ 将军!` - 当前方被将军
- `当前回合: ... - ✓ 将死! 黑方赢!` - 游戏结束，黑方获胜

## 技术细节 / Technical Details


### 浏览器要求 / Browser Requirements
- 支持 ES6 Modules
- Canvas 2D API
- 推荐: Chrome, Firefox, Safari, Edge (最新版本)

## 故障排除 / Troubleshooting

### 棋盘不显示 / Board doesn't show
- 检查浏览器控制台是否有错误消息
- 确保所有 JS 文件在 `js/` 目录中
- 尝试用本地 HTTP 服务器运行 (不要直接用 file://)

### 棋子无法移动 / Pieces can't move
- 点击棋子后确认有绿色方块出现
- 只能移动当前回合的棋子（红/黑）
- 检查该棋子是否真的有合法走法

### 游戏看起来不对齐 / Game doesn't align properly
- 可能是高 DPI 屏幕问题
- 尝试刷新页面
- 检查浏览器缩放是否为 100%

## 游戏提示 / Tips

1. 保护好你的将/王！
2. 利用河界限制对方象的活动
3. 记住国际象棋的兵可以升变为后，会很强大
4. 中国象棋的炮需要炮架才能吃子
5. 随时注意"将帅照面"规则
